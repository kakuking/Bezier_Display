Bezier Curves are a method to parametrically describe curves given an arbitrary number of "control points". It can easily be seen how this would be useful in animation (Describing movement smoothly using these curves) to game development (dedscribing paths or other things for the game).
Since we must describe an n-degree polynomial, we need n+1 constraints to be able to calculate the coefficients. In Bezier Curves, these are given with control points that describe which "direction" the curve must bend towards. In our program, these can be placed by left clicking. Further, these can be dragged around to change their location, and also, if the user presses the "d" key while hovering over one, it is deleted.
There must be at least 2 control points for a bezier curve to be rendered

Next, we must develop the surface of revolution of this bezier curve. To Define the axis of rotation, we can right click at 2 points, and that line will be used as the axis of rotation, next, it performs a full rotation of the bezier curve (sampled at 0.1 parameter steps).
